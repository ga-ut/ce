# @ga-ut/ce

Custom Element library with isolated state binding and an optional router.

## Usage

```ts
import { CE, html } from "@ga-ut/ce/web";
```

## Entry Points

- `@ga-ut/ce`
- `@ga-ut/ce/core`
- `@ga-ut/ce/web`
